# Blog App

Next.jsの課題を実装するためのアプリケーションです。課題の詳細は講師より提示されます。

## Getting Started

```shell-session
# /packages/ui
$ pnpm build # or pnpm dev
```

```shell-session
# /apps/blog-app
$ pnpm dev
```
